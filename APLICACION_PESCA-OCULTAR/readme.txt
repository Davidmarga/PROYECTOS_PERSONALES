Proyecto preparado para ejecutar con xammp o similar, importar la base de datos a phpmyadmin y después ejecutar el proyecto, pagina principal es index.php

Es un programa preparado para almacenar datos de participantes en competiciones de pesca, generación de clasificaciones según un sistema de puntuación preestablecido, rankings, y recuperación de esta información desde la base de datos
Desarrollado con php. JavaScript, HTML y css.
